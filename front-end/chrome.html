# Chrome

目前最流行，最强大的浏览器，也是前端开发必备的浏览器。下面我们来介绍使用 Chrome 浏览器的一些小技巧。

# 快捷键

老生常谈，任何常用的应用的快捷键都必须熟练使用，玩得 66 的, 下面介绍常用的快捷键，再次声明本文档不是大而全的文档，Chrome 快捷键组合大概有几十种，但我们只需要记住最常用的几种就行了

Chrome 快捷键和前面讲到的 iterm2, VS Code 快捷键几乎一样，事实证明优秀的设计是每个应用都会共同遵守的

  • cmd + T 打开一个新的 Tab
  • cmd + 数字键 快速跳转到和数字键对应的 Tab， cmd + 9 跳转到最后一个 tab
  • cmd + shift + N 以无痕模式打开新窗口
  • cmd + w 关闭当前窗口
  • cmd + N 打开一个新的浏览器窗口

# 插件

Chrome 的插件商店的强大程度无与伦比，下面来介绍我们开发中经常用到的一些插件第三方商店

  • 沙拉查词 必装的插件，划词翻译功能是我目前用过的最舒服的
  • Quick QR 必装的插件，将链接转换为二维码
  • lighthouse Google 官方出品，可以分析出你的网页的性能数据，以及列出可以优化的点
  • JSON Formatter 格式化 JSON 文件，让你在浏览器打开 JSON 文件时获得更愉悦的体验
  • React Developer Tools 开发 React 应用时用的辅助工具，大部分人应该都装了
  • Vue.js devtools 同上，开发 Vue 应用时的辅助工具
  • Octotree 方便你查看 github 项目结构
  • OctoLinker 查看 github 文件时可以一键打开引入文件的路径
  • Allow-Control-Allow-Origin 为不支持跨域请求的接口的响应头加 Allow-Control-Allow-Origin: *
  • 前端工具利器
  • Tampermonkey：用户脚本管理器（油猴）
  • Infinity 新标签页
  • Adblock Plus - 免费广告拦截插件
