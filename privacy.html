隐私政策

本隐私政策旨在向您说明华通云开发工具（以下简称 "本工具"）如何收集、使用、保护和披露您的个人信息。本工具致力于保护用户隐私，确保用户信息安全。

一、信息收集与使用

基本功能信息处理

本工具是一款二维码生成、加解密以及优化 network 的综合性工具。在使用过程中，仅针对部分网站进行解密操作。在执行这些功能时，我们不会主动收集您的任何个人身份信息，如姓名、地址、电话号码或电子邮件地址等。我们仅处理与二维码相关的数据以及为实现特定网站解密功能所必需的数据，这些数据在处理过程中不会与您的个人身份相关联。

HTML 生成相关信息

当您使用本工具生成 HTML 内容时，我们仅收集和处理您输入的用于生成 HTML 的参数和数据，这些数据仅用于在您的本地设备上完成 HTML 的生成操作，不会传输给第三方或用于其他任何目的。

二、信息存储

所有在本工具中处理的数据，包括用于二维码操作和特定网站解密的数据，都仅在您的本地设备上进行临时存储，以确保操作的高效性。当您关闭本工具后，这些临时数据将被自动清除，不会在我们的服务器或其他任何存储介质中保留。对于生成的 HTML 文件，其存储位置由您自行决定，我们不会对其进行存储或访问。

三、信息保护

安全措施

我们采用了先进的安全技术和严格的安全管理措施来保护您的数据。在本工具的设计和开发过程中，针对二维码生成、加解密以及特定网站解密功能，我们都进行了严格的安全测试，以确保数据在处理过程中的保密性、完整性和可用性。

解密限制

本工具只针对部分网站解密，这种限制是为了确保我们的操作在合法和安全的范围内进行。我们不会尝试对未授权的网站或内容进行解密，从而避免了因非法访问可能导致的信息泄露风险。

四、第三方信息共享

我们郑重承诺，不会将您的任何信息与第三方共享。无论是在二维码操作、特定网站解密还是 HTML 生成过程中，您的数据都将严格保密，仅在您的本地设备内进行处理，不会被传递给其他任何公司、组织或个人。

五、用户权利

您对您的数据拥有完全的控制权。如果您对本工具的数据处理方式有任何疑问或担忧，您可以随时停止使用本工具。我们不会对您的数据进行任何形式的保留或备份，您的数据在您的本地设备上的存储和处理完全由您自主决定。

六、隐私政策更新

我们可能会根据业务发展和法律法规的变化对隐私政策进行更新。当隐私政策发生重大变更时，我们将通过在本工具的官方网站或其他适当的渠道发布通知的方式告知您。请您定期查看隐私政策以确保您了解我们的最新信息处理方式。

请注意，使用本工具即表示您同意本隐私政策的条款。如果您不同意本隐私政策，请不要使用本工具。
